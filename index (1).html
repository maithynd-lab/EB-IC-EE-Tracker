Cần cấu hình Firebase để sync real-time

1. Vào console.firebase.google.com → Tạo project → Thêm Web app → Copy config
2. Mở file index.html → Tìm firebaseConfig → Dán config vào
3. Trong Firebase Console: Realtime Database → Tạo database → Rules → đặt ".read": true, ".write": true
4. Sau khi cấu hình xong, nhiều người mở link sẽ thấy update của nhau ngay lập tức ✨
